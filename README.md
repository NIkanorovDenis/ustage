# Ustage

Docker-окружение для локальной разработки и последующей настройки CI/CD сайта на «1С-Битрикс».

## Локальный запуск

1. Скопировать `.env.example` в `.env` и установить уникальные пароли.
2. Поместить файлы сайта в каталог `www/`.
3. При необходимости поместить SQL-дамп в `import/`.
4. Запустить окружение:

```bash
docker compose up -d --build
```

Локальный адрес: <http://localhost:8080>.

## Состав окружения

- Nginx
- PHP-FPM 8.1 с расширениями для Битрикса
- MySQL 8.0

`www/`, дампы, кеши, пользовательские загрузки и файлы с секретами не входят в первый инфраструктурный коммит. Перед включением CI/CD код проекта будет отделён от runtime-данных.
